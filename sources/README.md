# Official Sources

本目录记录官方资料入口。Agent 使用 RAG、GraphRAG 或第三方资料后，仍应尽量回到这些来源做最终核验。

| 来源 | 文件 | 用途 |
| --- | --- | --- |
| 3GPP Portal | `3gpp-portal.md` | 查询 TS/TR、CR、TDoc、会议、工作组和版本信息 |
| 3GPP FTP | `3gpp-ftp.md` | 下载原始规范、会议材料、TDoc 包、Meeting Report |
| 3GPP Forge | `3gpp-forge.md` | 查看规范文本版本、提交历史和 diff |

官方来源优先级高于第三方解释和模型总结。
