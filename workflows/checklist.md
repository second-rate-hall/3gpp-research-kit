# 3GPP Agentic Research Checklist

## 研究前

- [ ] 已明确研究问题。
- [ ] 已判断问题类型：clause 解释、CR 溯源、Release 对比、公司立场、功能演进、协议流程等。
- [ ] 已列出可能涉及的协议层、工作组、TS/TR、Release。
- [ ] 已确认优先使用官方来源或本地可信资料。

## 研究中

- [ ] 已检索相关 TS/TR。
- [ ] 已检查是否需要 CR、TDoc 中的会议材料或 Meeting Report。
- [ ] 已记录 evidence table。
- [ ] 已区分事实、推断和待核验内容。
- [ ] 已检查是否混淆 Release。
- [ ] 已检查是否混淆 TS/TR、CR、TDoc、Meeting Report 的层级。
- [ ] 已检查第三方解释是否仅作为辅助材料。

## 交付前

- [ ] 每个关键结论都有证据编号或来源说明。
- [ ] 无法确认的内容已标注为“待核验”。
- [ ] 报告中包含规范依据表。
- [ ] 报告中包含证据链或 evidence table。
- [ ] 报告中包含未确认点与后续核验建议。
- [ ] 没有把模型推断写成标准事实。

## 禁止行为

- [ ] 不得只基于模型常识回答复杂标准问题。
- [ ] 不得把 TDoc 和 TS/TR、CR、Meeting Report 简单并列。
- [ ] 不得把第三方博客、论坛或模型总结当作官方结论。
- [ ] 不得省略 Release / version / clause 的待核验标注。
